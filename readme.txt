# Data
These data contain reviews of 1312 arabica and 28 robusta coffee beans from the Coffee Quality Institute's trained reviewers. The features include:
# Quality Measures
1- Aroma
2- Flavor
3- Aftertaste
4-Acidity
5-Body
6- Balance
7- Uniformity
8-Cup Cleanliness
9- Sweetness
10- Moisture
11- Defects

# Bean Metadata
12-Processing Method
13- Color
14 Species (arabica / robusta)

# Farm Metadata
15-Owner
16- Country of Origin
17- Farm Name
18-Lot Number
19-Mill
20-Company
21-Altitude
22-Region
